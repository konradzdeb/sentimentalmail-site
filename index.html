Capabilities

Built for fast inbox use: clean text extraction, explainable scoring, and compact markers that appear where you read.

Local text extraction & cleaning

Extracts subject and body, removes HTML markup, normalises whitespace, and produces clean text for analysis.

Sentiment & tone scoring

Computes sentiment distributions (positive, neutral, negative) and derives higher-level signals.

Intent signals

Derives indicators such as urgency and emotional intensity, normalised for consistent visual comparison.

Deterministic message classification

Assigns discrete markers with logic designed to be explainable and configurable rather than hard-coded.

Inline Apple Mail integration

Renders results directly within the message view—no separate dashboard or companion app required.

Native SwiftUI rendering

Adaptive layouts that respect system font and size changes, with rendering structured for testing.

Customisation

Control how markers look and behave, with system-native preferences and accessibility in mind.

Marker styles

Toggleable marker semantics (flag-style vs colour-style labelling).

Adjustable font sizing

Supports accessibility and readability with user-adjustable sizing.

Localisation-ready

All user-facing strings are localisable and prepared for multi-language expansion.

Performance & reliability

Built to run quickly in high-volume inboxes, with stable, predictable behaviour.

Lightweight execution

Designed to analyse messages quickly without blocking UI rendering.

Deterministic behaviour

The same input produces the same output, without adaptive components that drift over time.

Structured for testing

Core logic and rendering are organised for unit and snapshot testing, emphasising stability across macOS versions.
